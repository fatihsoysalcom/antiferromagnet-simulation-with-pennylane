# Antiferromagnet Simulation with PennyLane

This Python script demonstrates simulating a simple two-spin antiferromagnet using PennyLane. It utilizes the Variational Quantum Eigensolver (VQE) algorithm to find the ground state energy and corresponding quantum state of an antiferromagnetic Hamiltonian.

## Language

`python`

## How to Run

1. Ensure you have PennyLane and NumPy installed (`pip install pennylane pennylane-qiskit`).
2. Run the script using `python antiferromagnet_simulation.py`.
